## 🎭 DELIVERY STYLE RULES

### 🔹 Pacing Rules

- **First sentence** = slow and deliberate
- **Middle section** = natural flow, no rush
- **Emotional lines** = slight slowdown
- **CTA** = gentle and reflective, not pushy

### 🔹 Pause Rules (CRITICAL FOR VIRALITY)

Add pauses manually in your script using ellipses:

```
"He thought… he was okay again…"
"But something felt off…"
"Not louder. Not worse. Just… familiar."
```

**👉 Use ellipses (…) for emotional tension**

These pauses:
- Create retention hooks (people want to know what's coming)
- Allow visuals to land harder
- Prevent "AI voice" robotic feeling
- Build emotional weight

### 🔹 Emotional Behavior

The voice should:
- Feel like **thinking**, not performing
- Sound **slightly tired but aware**
- **Never** sound excited or salesy
- **Hold emotional tension** in silence
