# Soulfables Viral Caption Bank

Your complete caption system for TikTok, Instagram, YouTube Shorts, and pinned comments.

## 🎯 CORE RULE

**First line = hook. Always.**

No explanation before curiosity.

## 🎬 TIKTOK CAPTION FORMULAS (HIGH RETENTION)

### 1. Identity Drift Hook
```
He thought he was okay again…

but something felt different this time.

Not worse.
Just… familiar.
```

### 2. Emotional Confusion Hook
```
Nothing changed on the outside.

But inside…
everything felt heavier.
```

### 3. Silent Breakdown Hook
```
Nobody noticed it.

But he stopped feeling like himself.
```

### 4. Relatable Pain Hook
```
You don't realize it at first…

until you don't recognize yourself anymore.
```

### 5. Minimal Viral Hook
```
He was still there…

just not really there.
```

## 🧠 VIRAL CAPTION RULES (CRITICAL)

- ✓ First line must create **curiosity gap**
- ✓ No long explanations upfront
- ✓ Use **short broken sentences**
- ✓ Add **emotional contrast** (fine vs not fine)
- ✓ End with **reflection or silence**
- ✓ Never sound like "marketing"
- ✓ Avoid emojis in first 2 lines (breaks retention)
- ✓ Let the story speak, not your brand

---

## 🔥 BONUS FORMULA (USE ANYWHERE)

**Hook → Emotional contrast → Subtle realization → Open-ended question**

Example:
```
Hook: "He thought he was okay again…"
Contrast: "But something felt off."
Realization: "Healing doesn't move in straight lines."
Question: "Have you felt this way?"
```

## 💡 Why This Works

People scroll past explanations. They stop for **curiosity**.

Your captions create a gap between what they see (fine on outside) and what they feel (something's wrong inside).

That gap = they keep watching.
