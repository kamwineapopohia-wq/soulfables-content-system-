# Soulfables Content System

Content engine for storytelling + social growth

## 🎯 Your Repeatable Content Engine

This is your system for building a data-backed growth machine, not just posting content randomly.

## 📁 Folder Structure

- **`scripts/`** - Write your content scripts here
- **`hooks/`** - Curated viral hooks that grab attention in 0-3 seconds
- **`captions/`** - Platform-specific captions (Instagram, TikTok, Twitter, LinkedIn)
- **`templates/`** - Reusable story templates following proven narrative structure
- **`analytics/`** - Track post performance to identify winning formulas

## 🚀 Daily Workflow

Every time you create content:

1. **Write script** → `/scripts`
2. **Pick hook** → `/hooks`
3. **Save captions** → `/captions`
4. **Track performance** → `/analytics`
